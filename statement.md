
Problem Statement

Keeping track of everyday expenses sounds simple, but in reality, most people either forget to write things down or feel overwhelmed by complicated budgeting apps. Small purchases like a coffee, a quick snack  often slip through the cracks. Because of this, people can lose track of how much theyâ€™re actually spending.
This project aims to solve that problem by offering a very simple, straightforward tool that lets users jot down their expenses.

Scope of the Project

* Adding expenses with a short note and an amount
* Automatically including the current date
* Showing all expenses in a clear list
* Calculating the total amount spent
* Handling mistakes like empty input or invalid numbers


Target Users

* Beginners learning Python, who want hands-on practice with functions, loops, and user input
* Students working on introductory coding projects
* Anyone who prefers a simple, no-nonsense way to track expenses without downloading apps
* People who like terminal tools and want something quick to run while working



High-Level Features

* Add new expenses by entering a description and amount
* Built-in date stamp so every entry is tagged with the current date
* View all expenses in a neatly formatted console list
* Instant total calculation for all recorded expenses
* Error handling for invalid or incomplete input
* Interactive menu that guides the user step by step
