# Vityarthi-Project

Topic Name: Pocket Expense Tracker

Overview :

The Pocket Expense Tracker is a simple Python program that helps you keep an eye on your daily spending. It runs in the terminal and lets you add new expenses, review everything and see how much you’ve spent in total. It’s designed to be easy to use, especially for beginners learning Python or anyone who wants a no-nonsense expense-tracking tool.

Features :

* Add expenses with description and amount
* View your expenses in a clean, organized list
* Instantly calculate your total spending
* Built-in error handling for invalid inputs
* Simple, interactive menu that’s easy to navigate

Technologies / Tools Used :

Python 3
Built-in modules:
 `datetime` for tracking dates
 Standard input/output for the interactive command-line interface

How to Install & Run :

1. Make sure Python 3 is installed on your computer.
2. Download the project files.
3. Open your terminal or command prompt.
4. Navigate to the folder where the script is located:

   ```bash
   cd path/to/project
   ```
5. Run the script by typing:

   ```bash
   python expense_tracker.py
   ```

How to Test the Program :

1. Start the program and choose 1to try adding an expense.
2. Enter a description and amount.
3. Choose 2 to see your expense in a list.
4. Select 3 to check that the total calculated
5. enter invalid values to see the error messages.
6. Use 4 to exit the application.
